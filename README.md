# Aquila Local

White-label reputation operations for U.S. marketing agencies. Operated remotely from the Philippines.

This is the public static website for Aquila Local. It includes the landing page, pricing section, and internal demo tools used to show the review operations workflow.

## Run

Open `index.html` in a browser.

## Deploy

The live Vercel project URL is `https://aquila-local-ops.vercel.app`.
